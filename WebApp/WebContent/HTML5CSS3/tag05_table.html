표 만들기

태그
table : 표의 시작
tr : 1줄 생성
td : 칸 생성
th : 강조하고 싶은 줄에 td대신에 th(글자가 칸의 가운데로 정렬, 글자가 진하게 설정)

column 단위로 처리
colgroup
col
span = 적용될 칸 크기

caption = 표의 제목 표시

thead : 표의 상단
tbody : 표의 내용
tfoot : 표의 하단

속성
width, height : px   or   %

표의 제목
첫번째 칸 두번째 칸 #ff0000 세번째 칸
두번째줄 첫번째 칸 두번째줄 두번째 칸
세번째줄 첫번째 칸 세번째줄 두번째 칸
